# 📖 About

Cricket Auction Simulator recreates the excitement of a professional T20 player auction directly in your browser.

Create your own franchises, strategically bid for players, manage your auction purse, retain star players, monitor auction statistics, and build your dream squad through a realistic auction experience.

Built using **HTML5**, **CSS3**, **JavaScript**, and **Tailwind CSS**, the simulator runs entirely inside your browser without requiring any installation.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/umarmahtab/cricket-auction-simulator.git
```

Navigate into the project

```bash
cd cricket-auction-simulator
```

Open

```text
index.html
```

in your preferred browser.

No installation or build tools are required.

---

# 🔊 Optional Sound Effects

Create a folder named

```text
sounds/
```

and add the following files:

```text
bid.wav
sold.wav
unsold.wav
tension.wav
crowd-loop.mp3
```

The simulator works perfectly without these files, but they significantly improve the auction atmosphere.
